# 🎮 康威生命游戏 (Conway's Game of Life)

一个现代化、交互式的康威生命游戏实现，使用原生 HTML、CSS 和 JavaScript 开发。

![Game of Life](https://img.shields.io/badge/version-1.0.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

## 📖 关于生命游戏

康威生命游戏是由英国数学家约翰·康威（John Conway）在1970年发明的细胞自动机。这是一个零玩家游戏，意味着游戏的演化完全由初始状态决定，不需要玩家进一步输入。

## 🎯 游戏规则

每个细胞会根据其周围8个相邻细胞的状态进行演化：

1. **生存**：如果一个活细胞周围有2或3个活细胞，它在下一代继续存活
2. **死亡**：如果一个活细胞周围的活细胞少于2个（孤独死亡）或多于3个（过度拥挤死亡），它在下一代死亡
3. **繁殖**：如果一个死细胞周围恰好有3个活细胞，它在下一代变成活细胞
4. **保持死亡**：其他情况下，死细胞保持死亡状态

## ✨ 功能特点

- 🎨 **现代化UI设计**：渐变色彩、流畅动画、响应式布局
- 🖱️ **交互式画布**：点击网格可手动添加/移除细胞
- ⚡ **可调节速度**：1-10级速度控制
- 📏 **可调节网格大小**：10-50像素的网格大小
- 🎲 **随机生成**：一键生成随机初始状态
- 📊 **实时统计**：显示当前代数和活细胞数量
- 🎨 **预设模式**：包含多种经典模式
  - 滑翔机 (Glider)
  - 闪烁灯 (Blinker)
  - 蟾蜍 (Toad)
  - 信标 (Beacon)
  - 脉冲星 (Pulsar)
  - 滑翔机枪 (Glider Gun)

## 🚀 快速开始

### 方法一：直接打开

1. 下载或克隆此仓库
2. 双击打开 `index.html` 文件
3. 开始游戏！

### 方法二：使用本地服务器

```bash
# 使用 Python 3
python -m http.server 8000

# 使用 Python 2
python -m SimpleHTTPServer 8000

# 使用 Node.js (需要安装 http-server)
npx http-server
```

然后在浏览器中打开 `http://localhost:8000`

## 🎮 使用说明

### 基本操作

1. **添加细胞**：点击网格上的任意位置切换细胞状态
2. **开始模拟**：点击"开始"按钮启动自动演化
3. **暂停**：点击"暂停"按钮停止演化
4. **重置**：点击"重置"按钮重置代数，保留当前模式
5. **清空**：点击"清空"按钮清空所有细胞
6. **随机生成**：点击"随机"按钮生成随机初始状态

### 调整设置

- **速度滑块**：调整演化速度（1-10）
- **网格大小滑块**：调整细胞大小（10-50像素）

### 尝试预设模式

点击预设模式按钮，快速加载经典生命游戏模式：

- **滑翔机**：会在网格上持续移动的小型模式
- **闪烁灯**：简单的两态振荡器
- **蟾蜍**：周期为2的振荡器
- **信标**：周期为2的振荡器
- **脉冲星**：周期为3的大型振荡器
- **滑翔机枪**：会持续产生滑翔机的复杂模式

## 🎯 核心实现

### 细胞状态更新

游戏使用双缓冲技术，维护两个网格数组：
- `grid`：当前代的细胞状态
- `nextGrid`：下一代的细胞状态

每一帧计算所有细胞的邻居数量，应用康威规则，然后交换两个网格。

### 边界处理

使用环形拓扑（toroidal topology），网格的左边缘连接到右边缘，上边缘连接到下边缘，创造无限平面的效果。

### 性能优化

- 使用 `requestAnimationFrame` 实现流畅动画
- 基于时间的更新循环，确保不同帧率下的一致速度
- Canvas 优化的渲染方式

## 🌟 有趣的模式

### 静态模式
- **方块**：2x2的正方形
- **蜂巢**：六边形结构

### 振荡器
- **闪烁灯**：周期为2
- **蟾蜍**：周期为2
- **信标**：周期为2
- **脉冲星**：周期为3

### 移动模式
- **滑翔机**：每4代移动一个对角格
- **轻量级飞船 (LWSS)**：每4代水平移动2格

### 复杂模式
- **滑翔机枪**：周期性产生滑翔机
- **彭塔十字花**：经过1103代后稳定

## 🔧 自定义开发

### 添加新的预设模式

在 `game.js` 的 `loadPattern` 方法中添加新模式：

```javascript
const patterns = {
    yourPattern: [
        [0, 1, 0],
        [0, 1, 0],
        [0, 1, 0]
    ]
};
```

然后在 `index.html` 中添加对应的按钮。

### 修改细胞颜色

在 `game.js` 的 `draw` 方法中修改渐变颜色：

```javascript
gradient.addColorStop(0, '#YOUR_COLOR_1');
gradient.addColorStop(1, '#YOUR_COLOR_2');
```
